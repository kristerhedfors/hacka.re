Welcome to hacka.re! hacka.re is a privacy-focused, serverless web-based LLM chat client that stores your API key, conversations, and settings locally in your browser.

hacka.re is serverless, meaning you can download the entire web page and run it offline, plus modify/extend it as you wish. hacka.re is published under the MIT No Attibution license, meant to be as free as freeware can be.

hacka.re supports creation of self-contained, strongly encrypted, password/session key-protected GPTs with individual system prompts and conversation data, which allows for sharing and collaboration over less secure channels.

To get started, you'll need to configure LLM access by specifying a base URL (OpenAI-compatible) and API key.

For more information about hacka.re, check out our About, Architecture, and Use Cases documentation.
